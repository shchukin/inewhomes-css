.anchor

This is JS based component. Making scrolling smooth.

See ../scripts/_anchor.js for details

The task is to catch HTML anchor (link targeting to the id=""), cancel the default browser behavior (quick jump to the anchor) and smoothly scroll to the target, manually.

  • Animating $('html, body')
  • Animation duration is relative to distance but not less than 300ms.
  • If there is a fixed header, then the distance to the target should be reduced equally to the height of the header.
  • Using data-anchor-gap="" we can manually add extra pixels to the distance.
  • If possible, we could try adding an easing function like this one: https://easings.net/#easeOutQuint but we may need a JS library for that.
  • The target can be any tag with id="": <div id="foobar">...</div>. Or, the empty link with the name="" attribute:
    <a name="foobar"></a> (href can be removed in this case).
